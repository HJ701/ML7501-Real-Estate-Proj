# Sample Data

This folder contains a tracked lightweight public sample so the repository can be run end-to-end without the full local raw snapshot.

Notes:

- The sample keeps the exact filenames expected by the source code.
- The transaction and rent files are deterministic subsets derived from the larger public datasets.
- The hotel file is kept in full because it is already small.
- The sample is intended for smoke tests, CLI verification, and instructor reproducibility checks.
- The sample is not intended to reproduce the headline full-data metrics in the README.
